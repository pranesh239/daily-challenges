## Built with:

- [Jest](https://jestjs.io/) : Javascript test library
- [Babel](https://babeljs.io/) : JavaScript compiler
- [ESLint](https://eslint.org/) : Javascript linter

## Steps

To install dependencies

```bash
yarn install --dev
```

To run test

```bash
yarn test
```

To run lint

```bash
yarn run lint
```

To run auto-fix from eslint

```bash
yarn run lint ./ --fix
```
